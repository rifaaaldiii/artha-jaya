Multi-Item Support

Satu nomor produksi dapat memiliki multiple item dengan jenis produksi, bahan, jumlah, dan harga yang berbeda.

👥

Role-Based Access

Setiap role memiliki akses status tertentu untuk memastikan workflow yang terstruktur dan aman.

📸

Progress Images

Upload foto progress produksi untuk dokumentasi dan tracking visual yang lebih baik.

📱

WhatsApp Notification

Notifikasi otomatis via WhatsApp saat produksi baru dibuat atau status diupdate.

💡 Key Feature

Status bergerak secara sequential - tidak bisa skip ke status berikutnya tanpa menyelesaikan status saat ini.

Flow Jasa & Layanan (Service)

Modul untuk mengelola jasa layanan pemasangan StepNosing dan Plint di lokasi pelanggan dengan scheduling dan assignment petugas.

Status Flow Sequence
Step 1
Jasa Baru
Admin Toko
Step 2
Terjadwal
Admin Toko / Kepala Teknisi
Step 3
Selesai Dikerjakan
Petugas / Kepala Teknisi
Step 4
Selesai
Admin Toko
👤

Customer Management

Integrasi dengan data pelanggan termasuk nama, kontak, dan alamat lokasi jasa.

📅

Scheduling System

Penjadwalan petugas dengan tanggal dan waktu pelaksanaan jasa di lokasi.

👷

Petugas Assignment

Multi-select petugas yang akan melaksanakan jasa di lapangan.

📸

Progress Image

Upload foto before/after dan during progress pengerjaan jasa.

💡 Key Feature

Petugas dapat update status "Selesai Dikerjakan", Admin Toko dapat finalisasi menjadi "Selesai".

3

User Roles & Permissions

Sistem memiliki role-based access control untuk memastikan setiap user hanya dapat mengakses dan mengupdate status sesuai tanggung jawabnya.

Role Produksi Access Jasa Access
Administrator Semua Status Semua Status
Admin Toko Produksi Baru → Selesai Jasa Baru → Terjadwal → Selesai
Admin Gudang Siap Produksi → Produksi Siap Diambil —
Kepala Teknisi Gudang Dalam Pengerjaan → Produksi Siap Diambil —
Kepala Teknisi Lapangan — Terjadwal → Selesai Dikerjakan
Petugas — Selesai Dikerjakan
4

WhatsApp Notification System

Integrasi WhatsApp API (Fonte) untuk notifikasi real-time ke team terkait dengan format pesan yang professional dan terstruktur.

🔔

Produksi Baru

Notifikasi ke team saat produksi baru dibuat dengan detail item produksi lengkap.

🔄

Update Status Produksi

Notifikasi saat status produksi berubah dengan informasi lengkap dan jelas.

🛠️

Jasa Baru

Notifikasi ke petugas saat ada jasa baru dengan detail pelanggan dan jadwal.

📊

Update Status Jasa

Notifikasi saat status jasa diupdate oleh petugas atau admin.

📝 Message Format

Professional formatting dengan sections, emoji, dan structured data untuk kemudahan pembacaan di WhatsApp.

5

Multi-Item System

Satu referensi nomor dapat memiliki multiple items untuk efisiensi dan organisasi data yang lebih baik.

🏭

Produksi Items

Setiap item memiliki: Jenis Produksi, Nama Bahan, Jumlah, Harga

🛠️

Jasa Items

Setiap item memiliki: Jenis Layanan, Jumlah, Harga

➕

Dynamic Add/Remove

Items dapat ditambahkan atau dihapus saat edit tanpa membuat record baru.

🔗

Cascade Delete

Menghapus parent record akan otomatis menghapus semua related items.
